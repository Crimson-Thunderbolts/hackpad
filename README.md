What does it do?

This is a 12 key hackpad that includes 12 keys an oled to display the functions of the keys and a rotating encoder to change the level this basicaly makes it act like a shift key changing the function of each key.

# **Hardware:**

MCU: Seeed XIAO RP2040

Switches:  12x Cherry MX compatible switches

Diodes: 12x Through-hole 1N4148 

Rotary Encoder: 1x EC11 Rotary encoder

Display: 1x 0.91 inch OLED display

# **BOM:**

Screws:  3x M3x16mm CounterSunk

Heatset Inserts: 3x M3x5mx4mm

MCU: Seeed XIAO RP2040

Switches:  12x Cherry MX compatible switches

Diodes: 12x Through-hole 1N4148 

Rotary Encoder: 1x EC11 Rotary encoder

Display: 1x 0.91 inch OLED display
